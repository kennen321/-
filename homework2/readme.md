#   堆積排序法(Heap Sort) 
  
## Definition(MAX HEAP) 
  
● 堆積的節點只有兩個子節點，從左到右排序下來 

● 在堆積排序還有很重要的一點是，父節點的數一定要比子節點大 

● 最大的數被存放在結構的最上方，即是樹根(root) 

● 樹根一定是最大值，將樹根(最大值)與最後一個節點調換， 

● 將最後一個節點(原樹根)取出，並加入已排序數列 

● 對整棵樹重新調整為最大堆積樹 

● 重複步驟依序可以列出答案
